# Bulut-Bilisim-Proje-2
Real-time IoT Data Streaming with AWS
# 3522 Bulut Bilişim Dersi - Proje 2
## Gerçek Zamanlı Veri Akışı ve İşleme

### Proje Özeti
Bu projede, sanal bir hava durumu sensöründen üretilen verilerin MQTT protokolü kullanılarak 
AWS IoT Core üzerinden alınması, AWS Kinesis ile işlenmesi ve DynamoDB üzerinde depolanması hedeflenmektedir.

### Kullanılacak Teknolojiler
- **Backend:** Python (Boto3 kütüphanesi)
- **Protokol:** MQTT / WebSocket
- **Bulut:** AWS (IoT Core, Kinesis, Lambda, DynamoDB)
